📐 CSS Spacing Examples

Interactive examples demonstrating margin, padding, and gap properties

Margin

Space outside elements - creates separation between elements

Box 1 - margin-bottom: 20px
Box 2 - margin-bottom: 20px
Box 3 - no margin
.box { margin-bottom: 20px; }

Padding

Space inside elements - creates breathing room for content

Box with padding: 20px
Box with padding: 40px
.box { padding: 20px; } .box-large { padding: 40px; }

Gap Property

Space between flex or grid items - modern alternative to margins

Item 1
Item 2
Item 3
.container { display: flex; gap: 20px; }

Margin Collapse

Adjacent margins combine into a single margin (larger value wins)

Box 1 - margin-bottom: 30px
Box 2 - margin-top: 20px

Result: 30px gap (larger margin wins)

/* Margins collapse - only 30px space between boxes */ .box1 { margin-bottom: 30px; } .box2 { margin-top: 20px; }

Auto Margin for Centering

Using margin: auto to center block elements horizontally

Centered with margin: 0 auto
.box { width: 200px; margin: 0 auto; }

Spacing System with CSS Variables

Using CSS custom properties for consistent spacing

Small padding
Medium padding
Large padding
:root { --spacing-xs: 4px; --spacing-sm: 8px; --spacing-md: 16px; --spacing-lg: 24px; --spacing-xl: 32px; } .box { padding: var(--spacing-md); }

Shorthand Properties

Different ways to write margin and padding

/* All sides */ margin: 20px; padding: 20px; /* Top/bottom, left/right */ margin: 10px 20px; padding: 10px 20px; /* Top, left/right, bottom */ margin: 10px 20px 30px; padding: 10px 20px 30px; /* Top, right, bottom, left (clockwise) */ margin: 10px 20px 30px 40px; padding: 10px 20px 30px 40px;
