🎨 CSS Flexbox Examples

Interactive demonstrations of common Flexbox patterns

1. Center Anything

The most common Flexbox use case - center content both horizontally and vertically.

.container { display: flex; justify-content: center; align-items: center; }
Live Demo
Centered box

2. Equal-Width Columns

Create columns that are equal width regardless of their content size.

.container { display: flex; gap: 16px; } .column { flex: 1; }
Live Demo
Short content
A column with quite a bit more text in it, but it still stays exactly the same width as its siblings.

3. Fixed Sidebar + Fluid Content

A sidebar with fixed width that never grows or shrinks, with content that fills remaining space.

.container { display: flex; gap: 16px; } .sidebar { flex: 0 0 200px; } .content { flex: 1; }
Live Demo
Fixed 200px sidebar — never grows, never shrinks
Content fills whatever space is left, and reflows as the window resizes

4. Sticky Footer

Footer pinned to bottom even on short pages using flexbox column layout.

html, body { height: 100%; margin: 0; } .page { display: flex; flex-direction: column; min-height: 100%; } .main { flex: 1; }
Live Demo
Header
Short page content. The footer stays pinned to the bottom because .main has flex: 1 and consumes all leftover vertical space.
Footer — always pinned to the bottom

5. Navbar Layout

Classic navigation bar with logo on left and links on right.

.navbar { display: flex; justify-content: space-between; align-items: center; }

6. Responsive Wrapping Cards

Cards that wrap onto new lines based on available space.

.container { display: flex; flex-wrap: wrap; gap: 16px; } .card { flex: 1 1 200px; }
