🎯 CSS Grid Examples

Interactive demonstrations of common Grid patterns

1. Basic Grid Layout

Fixed column with flexible columns that share remaining space.

.grid { display: grid; grid-template-columns: 200px 1fr 1fr; grid-template-rows: 80px auto; gap: 16px; }

2. Responsive Card Grid

Auto-fitting cards with no media queries - cards automatically reflow based on available space.

.card-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); gap: 16px; }

3. Page Shell Layout

Complete page layout with header, sidebar, main content, and footer using grid-template-areas.

.page { display: grid; grid-template-columns: 200px 1fr; grid-template-rows: auto 1fr auto; grid-template-areas: "header header" "sidebar main" "footer footer"; min-height: 100vh; }

4. Center Content

Perfect centering using place-items shorthand.

.container { display: grid; place-items: center; min-height: 100vh; }
