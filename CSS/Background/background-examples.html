🎨 CSS Background Examples

Interactive examples demonstrating CSS background properties and gradients

Background Color

Different color formats for background colors

Solid Hex
RGBA (50% opacity)
HSLA (70% opacity)
.solid { background-color: #667eea; } .rgba { background-color: rgba(102, 126, 234, 0.5); } .hsla { background-color: hsla(230, 80%, 66%, 0.7); }

Background Size

How background images scale within elements

Cover - fills entire element
Contain - fits within element
Stretch - 100% 100%
.cover { background-size: cover; } .contain { background-size: contain; } .stretch { background-size: 100% 100%; }

Gradients

Different types of CSS gradients

Linear Gradient
Radial Gradient
Conic Gradient
.linear { background: linear-gradient(to right, #667eea, #764ba2); } .radial { background: radial-gradient(circle, #667eea, #764ba2); } .conic { background: conic-gradient(#667eea, #764ba2, #667eea); }

Gradient Patterns

Creating patterns with CSS gradients

Striped Pattern
Dotted Pattern
.stripes { background: repeating-linear-gradient( 45deg, #667eea, #667eea 10px, #764ba2 10px, #764ba2 20px ); } .dots { background: radial-gradient(circle, #667eea 2px, transparent 2px); background-color: #764ba2; background-size: 20px 20px; }

Gradient Text

Applying gradients to text using background-clip

Gradient Text
.gradient-text { background: linear-gradient(to right, #667eea, #764ba2); -webkit-background-clip: text; background-clip: text; color: transparent; }

Background Shorthand

Combining multiple background properties

/* Full shorthand */ background: #ffffff url('image.jpg') no-repeat center / cover; /* Breakdown */ background-color: #ffffff; background-image: url('image.jpg'); background-repeat: no-repeat; background-position: center; background-size: cover;
