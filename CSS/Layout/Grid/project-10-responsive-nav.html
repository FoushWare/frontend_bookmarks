Project 10: Responsive Navigation

Project Statement

Build a responsive navigation that adapts from horizontal to vertical layout based on viewport width. This project teaches you how to create flexible navigation patterns without media queries.

Requirements:

  • Create a navigation with logo and links
  • Use CSS Grid with auto-fit for responsiveness
  • Links should have minimum width of 100px
  • Add 16px gap between items
  • Style with appropriate colors
  • Observe how it adapts on smaller screens
