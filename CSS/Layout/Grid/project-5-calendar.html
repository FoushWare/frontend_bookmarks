Project 5: Calendar Layout

Project Statement

Build a calendar grid with proper day alignment and event spanning across multiple days. This advanced project teaches you complex grid placement with named lines and spanning.

Requirements:

  • Create a 7-column grid for days of the week
  • Add day headers (Mon, Tue, Wed, etc.)
  • Create calendar cells for dates
  • Add an event that spans 3 days
  • Use grid-column for event spanning
  • Add 8px gap between cells
  • Style with appropriate colors
