Inheritance (@extend)

Share styles between selectors

// SCSS .button-base { padding: 12px 24px; border: none; border-radius: 4px; cursor: pointer; margin: 5px; } .button-primary { @extend .button-base; background: #667eea; color: white; } .button-secondary { @extend .button-base; background: #764ba2; color: white; }

Maps

Store key-value pairs for organized data

Light Theme

Background: #ffffff

Text: #333333

Dark Theme

Background: #1a1a2e

Text: #eaeaea

// SCSS $themes: ( light: ( bg: #ffffff, text: #333333 ), dark: ( bg: #1a1a2e, text: #eaeaea ) ); @mixin theme($theme-name) { $theme: map-get($themes, $theme-name); background: map-get($theme, bg); color: map-get($theme, text); } .light-theme { @include theme('light'); } .dark-theme { @include theme('dark'); }

Conditionals

Use @if, @else if, and @else for conditional logic

Success State

This element uses conditional styling based on state

// SCSS @mixin alert-style($type) { padding: 20px; border-radius: 8px; @if $type == 'success' { background: #27ae60; color: white; } @else if $type == 'error' { background: #e74c3c; color: white; } @else if $type == 'warning' { background: #f39c12; color: white; } @else { background: #3498db; color: white; } } .success-alert { @include alert-style('success'); }
