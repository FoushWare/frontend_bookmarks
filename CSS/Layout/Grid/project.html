Project: Responsive Card Grid

Create a responsive card grid that automatically adjusts the number of columns based on the viewport width. Cards should have a minimum width of 200px and share available space equally.

Requirements:

  • Use CSS Grid for the layout
  • Cards should be at least 200px wide
  • Cards should share available space equally
  • No media queries required
  • Add a 16px gap between cards
  • Style cards with a light background and border
