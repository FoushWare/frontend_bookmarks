Project 3: Dashboard Grid

Project Statement

Build a complex dashboard layout with widgets of different sizes using line-based placement. This project teaches you how to use grid-column and grid-row to create asymmetrical layouts common in dashboard designs.

Requirements:

  • Create a 3-column grid layout
  • Make the first widget span 2 columns and 2 rows (featured)
  • Other widgets should be 1x1 size
  • Use grid-column and grid-row for placement
  • Add 16px gap between widgets
  • Style widgets with different colors
  • Make the layout responsive
