# CSS Grid

A complete masterclass: two-dimensional layout in CSS, from first principles to production
architecture decisions.

## Prerequisites
CSS box model, selectors/specificity, Flexbox fundamentals, basic media queries.

## Estimated time
~12–16 hours across all four stages.
