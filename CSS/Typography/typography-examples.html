📝 CSS Typography Examples

Interactive examples demonstrating CSS typography properties

Font Size

Different font sizes using relative units

Extra Small - 0.75rem (12px)

Small - 0.875rem (14px)

Base - 1rem (16px)

Large - 1.125rem (18px)

Extra Large - 1.25rem (20px)

2X Large - 1.5rem (24px)

.text-xs { font-size: 0.75rem; } .text-sm { font-size: 0.875rem; } .text-base { font-size: 1rem; } .text-lg { font-size: 1.125rem; } .text-xl { font-size: 1.25rem; } .text-2xl { font-size: 1.5rem; }

Different font weights for emphasis

Medium (500)

Semi-bold (600)

Bold (700)

Black (900)

.weight-400 { font-weight: 400; } .weight-500 { font-weight: 500; } .weight-600 { font-weight: 600; } .weight-700 { font-weight: 700; } .weight-900 { font-weight: 900; }

Line Height

Line height affects readability and spacing between lines

Line height 1 - Tight spacing. This text has very little space between lines, which can make it harder to read for longer passages of text.

Line height 1.5 - Comfortable spacing. This text has good spacing between lines, making it easier to read for body text and longer content.

Line height 2 - Loose spacing. This text has generous spacing between lines, which can be useful for certain design styles or accessibility.

.lh-1 { line-height: 1; } .lh-1-5 { line-height: 1.5; } .lh-2 { line-height: 2; }

Letter Spacing

Adjusting space between characters

Normal letter spacing

Tight letter spacing (-0.5px)

Wide letter spacing (2px)

.ls-normal { letter-spacing: normal; } .ls-tight { letter-spacing: -0.5px; } .ls-wide { letter-spacing: 2px; }

Text Alignment

Different text alignment options

Left aligned text - the default alignment for most content

Center aligned text - useful for headings and special content

Right aligned text - useful for certain design patterns

Justified text - both left and right edges are aligned, creating equal spacing between words. This can sometimes create uneven spacing between words.

.align-left { text-align: left; } .align-center { text-align: center; } .align-right { text-align: right; } .align-justify { text-align: justify; }

Text Overflow (Single Line)

Truncating text with ellipsis for single line

This is a very long text that will be truncated with an ellipsis when it exceeds the container width.
.truncate { white-space: nowrap; overflow: hidden; text-overflow: ellipsis; max-width: 300px; }

Line Clamp (Multiline)

Truncating text after multiple lines

This is a longer text that will be truncated after three lines. The line-clamp property allows you to limit the number of lines displayed and add an ellipsis. This is useful for card descriptions, article previews, and any content that needs to be constrained to a specific number of lines while maintaining readability.
.clamp { display: -webkit-box; -webkit-line-clamp: 3; -webkit-box-orient: vertical; overflow: hidden; }
