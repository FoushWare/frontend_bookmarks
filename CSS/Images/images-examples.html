🖼️ CSS Images Examples

Interactive examples demonstrating CSS image properties and filters

Object-Fit

How images fit within their containers

.fill { object-fit: fill; } .contain { object-fit: contain; } .cover { object-fit: cover; }

Image Filters

Applying visual effects to images

.grayscale { filter: grayscale(100%); } .blur { filter: blur(3px); } .brightness { filter: brightness(150%); } .contrast { filter: contrast(150%); }

Opacity

Controlling image transparency

.opacity-100 { opacity: 1; } .opacity-75 { opacity: 0.75; } .opacity-50 { opacity: 0.5; } .opacity-25 { opacity: 0.25; }

Responsive Image

Image that scales with viewport width

img { max-width: 100%; height: auto; border-radius: 8px; }

Circular Avatar

Creating circular profile images

.avatar { width: 100px; height: 100px; border-radius: 50%; object-fit: cover; }
