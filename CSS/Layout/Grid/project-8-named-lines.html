Project 8: Named Lines Layout

Project Statement

Build a layout using named grid lines for semantic and maintainable code. Named lines make your CSS more readable and easier to understand at a glance.

Requirements:

  • Create a layout with full-bleed sections and constrained content
  • Use named lines for the grid template
  • Name lines: full-start, content-start, content-end, full-end
  • Place hero section to span full width
  • Place article in constrained content area
  • Add appropriate styling
