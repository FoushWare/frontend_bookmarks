Interactive examples demonstrating responsive design techniques

Fluid Typography

Text that scales smoothly with viewport width

This text uses clamp() - scales between 1rem and 2rem

This text uses calc() - 16px + 0.5vw

.clamp-text { font-size: clamp(1rem, 3vw, 2rem); } .calc-text { font-size: calc(16px + 0.5vw); }

Viewport Units

Sizing elements relative to viewport dimensions

50vw width
20vh height
.vw-box { width: 50vw; margin: 0 auto; } .vh-box { height: 20vh; }

Responsive Container

Container that scales with viewport but has max width

This container is 100% wide but max 600px

.container-demo { width: 100%; max-width: 600px; margin: 0 auto; }

CSS Functions

Using clamp, min, max, and calc for responsive values

/* Clamp for responsive typography */ font-size: clamp(16px, 2vw, 24px); /* Min for responsive width */ width: min(100%, 800px); /* Max for responsive height */ height: max(300px, 50vh); /* Calc for calculations */ padding: calc(1rem + 2px); width: calc(100% - 40px);
